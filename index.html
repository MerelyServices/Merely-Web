merely

merely is an open source discord bot intended to be highly modular and able to enhance a discord server it's invited to with useful commands like...

  • m/poll to bring live polls to discord
  • m/welcome and m/farewell for custom server leave/join messages
  • m/clean to quickly clean up a channel
  • m/language to use translations of the bot to other languages
  • home (current)
  • try merely
  • add merely to your server

commands

  • to use a merely command, start your command with merely , @merely or m/. for example; merely help, m/help or @merely help
  • the syntax for a command is like so: merely command (required1 | required2) [optional] where | means or.
