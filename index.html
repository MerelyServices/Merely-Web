Merely-Framework

Merely is an open source Discord bot framework. The example implementation of Merely-Framework (Merelybot) uses all of the included extensions for demonstration. Here is some highlighted commands;

  • /poll to bring live polls to Discord
  • /welcome and /farewell for custom server join/leave messages
  • /clean to quickly delete bot messages from a channel
  • /language to speak with the bot in another language
  • Home (current)
  • Try MerelyBot
  • Add MerelyBot to your server

Commands

  • To use a merely command, start your command with / and choose Merely's command from the list.
